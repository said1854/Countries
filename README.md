# Countries

Countries is a web application for exploring Countries.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
git clone
cd countries
npm install ~ yarn add
npm run dev ~ yarn dev
```
